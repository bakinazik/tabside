Changelog

  • The list view has been redesigned with a modern design approach.
  • You can now create categories using hashtags anywhere in your note. Created categories appear at the top of the main note list, and you can rearrange them by dragging and dropping.
  • Some known issues have been resolved.
  • The interface has been modularized with at least 20% performance improvement.
  • Added support for some basic Markdown syntax in the editor.
  • The block exit command for pre and blockquote formats has been changed to Shift + Enter.
  • The search bar now focuses automatically, allowing faster and smoother note searches.
  • Toolbar buttons now change color according to the elements in the editor.
  • Insert image and video. (YouTube, Dailymotion, Vimeo or MP4)
  • Intuitive notifications added for critical operations.
  • The note export feature has been updated. Notes can now be exported as HTML, Markdown, or plain text (TXT).
  • Bug fixes and other improvements.
